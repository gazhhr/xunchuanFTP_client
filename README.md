# Android FTP Client

A lightweight and easy-to-use FTP client for Android devices.  
Connect to FTP servers and manage files directly from your phone.

一个轻量级、易于使用的 Android FTP 客户端。  
让你可以直接在安卓手机上连接 FTP 服务器并管理文件。

---

# Features / 功能特点

- 📂 Upload and download files  
  文件上传与下载

- 🌐 Connect to FTP servers  
  连接 FTP 服务器

- ⚡ Fast and lightweight  
  轻量快速

- 📱 Android-only support  
  仅支持 Android 系统

- 🔒 Username & password authentication  
  支持用户名密码认证

- 📡 Local network  
  支持局域网访问
